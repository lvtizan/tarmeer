# 阿里云 ECS 数据库与环境配置

## 当前使用的数据库

- **类型**：MySQL（5.7+ 或 8.0，建议 8.0）
- **数据库名**：`tarmeer`
- **依赖表**：`designers`、`projects`、`contacts`

后端通过环境变量连接数据库，见下表。

---

## 一、在阿里云服务器上安装 MySQL（若未安装）

```bash
# CentOS / Aliyun Linux
sudo yum install -y mysql-server
sudo systemctl start mysqld
sudo systemctl enable mysqld

# 首次安装后获取临时 root 密码（仅部分版本）
sudo grep 'temporary password' /var/log/mysqld.log
# 登录后修改密码：ALTER USER 'root'@'localhost' IDENTIFIED BY '你的强密码';
```

若使用 Ubuntu：

```bash
sudo apt update
sudo apt install -y mysql-server
sudo systemctl start mysql
sudo systemctl enable mysql
sudo mysql_secure_installation
```

---

## 二、创建数据库与表

1. 登录 MySQL（root 或已有管理员账号）：

   ```bash
   mysql -u root -p
   ```

2. 执行初始化脚本（部署后服务器上已有 `schema` 目录）：

   ```bash
   # SSH 登录后，后端部署目录为 /tarmeer/tarmeer_api
   cd /tarmeer/tarmeer_api
   mysql -u root -p < schema/init.sql
   ```
   或使用脚本（会提示输入 MySQL 密码）：  
   `bash schema/init-db.sh`

   该脚本会：
   - 创建数据库 `tarmeer`（不存在时）
   - 创建表：`designers`、`projects`、`contacts`

3. 若希望单独使用数据库账号（不用 root），可创建用户并授权：

   ```sql
   CREATE USER 'tarmeer'@'localhost' IDENTIFIED BY '设置一个强密码';
   GRANT ALL PRIVILEGES ON tarmeer.* TO 'tarmeer'@'localhost';
   FLUSH PRIVILEGES;
   ```

   之后 `.env` 里使用 `DB_USER=tarmeer`、`DB_PASSWORD=设置一个强密码`。

---

## 三、后端 .env 配置（/tarmeer/tarmeer_api/.env）

在服务器上编辑 `/tarmeer/tarmeer_api/.env`，保证与当前数据库一致：

```env
# 数据库（本机 MySQL）
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=你的MySQL密码
DB_NAME=tarmeer

# JWT（生产必填，建议 32 位以上随机字符串）
JWT_SECRET=你的JWT密钥

# 服务
PORT=3002
NODE_ENV=production

# 前端域名（用于 CORS、邮件链接等）
FRONTEND_URL=http://designer.tarmeer.com

# SMTP（发验证邮件等，可选；不配则注册成功但收不到邮件）
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=你的邮箱
SMTP_PASS=应用专用密码
SMTP_FROM=noreply@tarmeer.com
```

- **DB_NAME** 必须为 `tarmeer`（与 `init.sql` 中一致）。
- **JWT_SECRET** 生产环境必填。
- SMTP 不配置时，注册接口仍会返回成功，但验证邮件不会发出。

---

## 四、重启后端并验证

```bash
pm2 restart tarmeer-api
pm2 logs tarmeer-api
```

然后测试注册或访问：

```bash
curl -s http://127.0.0.1:3002/api/health
```

返回 JSON 且无报错即表示环境与数据库配置正确。

---

## 五、小结

| 项目     | 值 |
|----------|-----|
| 数据库类型 | MySQL |
| 数据库名   | **tarmeer** |
| 表        | designers, projects, contacts |
| 初始化脚本 | `server/schema/init.sql` |
| 环境变量   | DB_HOST, DB_USER, DB_PASSWORD, **DB_NAME=tarmeer**, JWT_SECRET, FRONTEND_URL, (SMTP_*) |

线上若尚未建库，在阿里云服务器上执行一次 `mysql -u root -p < server/schema/init.sql`，并按要求配置 `/tarmeer/tarmeer_api/.env` 即可。
